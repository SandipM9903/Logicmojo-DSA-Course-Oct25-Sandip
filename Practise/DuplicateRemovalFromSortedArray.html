Duplicate Removal Algorithm Visualizer

This visualization demonstrates how the optimized algorithm removes duplicates from a sorted array in-place.

How the Algorithm Works

  • The algorithm uses two pointers: i (slow pointer) and j (fast pointer)
  • i tracks the position of the last unique element
  • j scans through the array looking for new unique elements
  • When a new unique element is found at position j, it's moved to position i+1
  • The final length is i+1, representing the number of unique elements

Algorithm Code

public static int removeDuplicates(int[] nums) {
if (nums.length == 0) return 0;
int i = 0;
for (int j = 1; j < nums.length; j++) {
if (nums[j] != nums[i]) {
i++;
nums[i] = nums[j];
}
}
return i + 1;
}

Time & Space Complexity Analysis

Time Complexity

O(n)

The algorithm uses a single loop that iterates through the array once with pointer j.

Each element is compared at most once, and unique elements are moved to their correct positions.

Space Complexity

O(1)

The algorithm operates in-place, modifying the input array without using additional data structures.

Only a constant amount of extra space is used for variables i and j.

Our Algorithm: O(1)
